# Artistify
Consumes Youtube Data API and provides a new interface for playing and queing music.

## EXPRESS

### install express

`npm i`

### run express server

`npm start`

### run server tests

while server is running,
`npm test`

# REACT

### enter react folder

`cd frontend`

### install react

inside of `frontend` folder,
`npm i`

### start dev server

inside of `frontend` folder,
`npm start`

### build react into static asset in public folder

inside of `frontend` folder,
`npm run build`
